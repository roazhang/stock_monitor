# 📊 零股庫存監控工具

多帳戶零股庫存比對、建議清單追蹤、批次下單範本一站搞定。
純前端網頁，不需安裝、不需伺服器，直接瀏覽器開啟即用。

🌐 **線上使用：** `https://你的帳號.github.io/stock_monitor/`

---

## ✨ 功能總覽

| 功能 | 說明 |
|------|------|
| 🔍 股票查詢 | 輸入代號或名稱，即時查詢各帳戶持有數量 |
| 📌 建議清單比對 | 上傳任意格式清單，自動比對各帳戶缺少哪些 |
| 📋 比對矩陣 | 所有股票 × 所有帳戶，一張表看清持有狀況與數量 |
| 🔴 缺少清單 | 各帳戶缺少的股票一覽，一鍵加入下單 |
| 👤 各帳戶 | 單一帳戶的缺少清單 |
| 📋 批次下單範本 | 支援永豐 CSV 與國泰 XML，從缺少清單一鍵產生 |

---

## 🚀 使用流程

### 第一步：上傳庫存檔案

將各帳戶的庫存檔案拖曳或點擊上傳，支援**一次上傳多個帳戶**。

**支援格式：**
- 永豐 `.xlsx`：需包含 `商品`（如 `1101 台泥`）與 `今餘` 欄位
- 國泰 `.csv`：需包含 `股票名稱(代號)`（如 `台泥(1101)`）與零股今餘欄位
- 其他含股票代號的 `.xlsx` / `.xls` / `.csv` 亦可

> ⚠️ `.xlsx` 請勿轉存為其他格式，程式直接讀取原始檔案。

---

### 第二步：上傳建議清單（選填）

點擊「📌 上傳建議清單」，上傳任意格式的推薦買進清單。

**欄位選擇器：**
上傳後程式自動掃描所有欄位，顯示每欄的樣本值與代號數量，讓你**點選含有股票代號的欄位**後確認。

- 格式不限：直行、橫排、多欄並列皆可
- 清單有多段分區也能完整抓取
- 下次換新清單只要重新上傳，不需修改程式
- 已上傳後可點 ⚙ **重新選欄**，無需重新上傳

---

### 第三步：開始比對

點「🔍 開始比對」，上方會顯示各帳戶的**持有股票數**與**缺少數量**摘要卡。

---

### 第四步：查看各頁籤結果

#### 🔍 股票查詢
- 輸入股票代號（如 `2330`）或名稱（如 `台積電`）即時搜尋
- 顯示每個帳戶的**持有股數**，未持有顯示「未持有」

#### 📌 建議清單比對（需上傳建議清單）
- 顯示建議清單總支數，及各帳戶已持有 / 缺少數
- 完整對照表：每支建議股 × 各帳戶是否持有
- 各帳戶的缺少清單，旁邊有 📋 按鈕可直接加入下單

#### 📋 比對矩陣
- 全部股票 × 全部帳戶一覽表，格子內顯示實際持有股數
- 可輸入關鍵字篩選，可切換「只看缺少」快速定位

#### 🔴 缺少清單
- 各帳戶尚未持有的股票列表
- 每筆旁邊有 📋 圖示可單支加入下單
- 右上方「📋 全部加入下單」可一次帶入所有缺少的股票

#### 👤 各帳戶頁籤
- 點選特定帳戶，只看該帳戶的缺少清單

---

### 第五步：產生批次下單（選填）

點右上角「📋 批次下單範本」按鈕開啟，或在缺少清單直接點「📋 加入下單」自動帶入。

---

## 📋 批次下單範本

支援兩家券商，切換頁籤選擇：

### 永豐 CSV

| 步驟 | 說明 |
|------|------|
| 1 | 點「📄 上傳永豐下單範本自動抓帳號」，從既有範本自動抓取所有帳號 |
| 2 | 或手動輸入帳號後按「＋ 新增」（可新增多個） |
| 3 | 在下方輸入股票代號，支援空格 / 逗號 / 換行混貼 |
| 4 | 點 CSV 預覽框全選，或按「📋 一鍵複製」 |
| 5 | 貼入永豐「批次下單」介面執行 |

**輸出格式：** 每個帳號 × 每支股票各產生一行 CSV，格式與永豐範本完全一致。

---

### 國泰 XML

| 步驟 | 說明 |
|------|------|
| 1 | 點「📄 上傳國泰批次範本自動填入帳號」，自動讀取 DealAcc / DealAccUI / SSID |
| 2 | 或手動填入三個欄位 |
| 3 | 輸入股票代號 |
| 4 | 點「📋 一鍵複製 XML」或「⬇ 下載 XML」 |
| 5 | 匯入國泰下單系統 |

**支援編碼：** 自動偵測 XML 宣告（`encoding="big5"` 或 `encoding="UTF-8"`），兩種格式皆可正確讀取，中文姓名不會亂碼。

> 💡 從「缺少清單」或「建議清單」點擊 📋 加入的股票代號，會**同時帶入永豐和國泰**兩個清單。

---

## 🛠 支援格式說明

### 庫存檔案（XLSX）

| 欄位 | 說明 |
|------|------|
| `商品` | 股票代號 + 名稱，如 `1101 台泥` |
| `今餘` | 持有股數 |

### 庫存檔案（CSV）

| 欄位 | 說明 |
|------|------|
| 欄 0：`股票名稱` | 格式：`台泥(1101)` |
| 欄 14：零股今餘 | 持有股數 |

### 建議清單

不限格式，只要有包含 4～6 位數字的股票代號欄即可。程式會：
1. 自動掃描全部列（含多段分區）
2. 顯示候選欄位供使用者點選
3. 自動嘗試抓取相鄰欄的股票名稱
