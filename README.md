# 📊 零股庫存監控工具

多帳戶零股庫存比對、建議清單追蹤、批次下單範本一站搞定。
純前端網頁，不需安裝、不需伺服器，直接瀏覽器開啟即用。

🌐 **線上使用：** `https://roazhang.github.io/stock_monitor/`

---

## ✨ 功能總覽

### 📂 庫存比對
- 一次上傳多個帳戶庫存（.xlsx / .csv）
- 自動比對哪些帳戶缺哪些股票
- 支援永豐、國泰等各券商匯出格式

### 🔍 股票查詢
- 輸入代號或名稱即時搜尋
- 顯示各帳戶持有數量

### 📌 建議清單比對
- 上傳任意格式的建議買進清單
- 自行選擇代號欄位（不限格式）
- 比對各帳戶已持有／尚未買入的建議股

### 📋 批次下單範本
支援兩家券商，直接從缺少清單一鍵帶入代號：

| 券商 | 格式 | 操作 |
|------|------|------|
| 永豐 | CSV | 複製後貼入永豐批次下單介面 |
| 國泰 | XML | 下載後匯入國泰下單系統 |

---

## 🚀 使用流程

### 第一步：上傳庫存
拖曳或點擊上傳各帳戶庫存檔案，支援一次多檔。

**支援格式：**
- 永豐 .xlsx（欄位：`商品`、`今餘`）
- 國泰 .csv（欄位：`股票名稱(代號)`、零股今餘）
- 其他含股票代號的 .xlsx / .csv

### 第二步：上傳建議清單（選填）
上傳後會自動掃描所有欄位，點選含有股票代號的欄位確認即可。
**格式不限，換新清單只要重新選欄，不需改程式。**

### 第三步：開始比對
點「🔍 開始比對」，結果分頁籤顯示：

| 頁籤 | 內容 |
|------|------|
| 🔍 股票查詢 | 輸入代號或名稱，查詢各帳戶持有數量 |
| 📌 建議清單比對 | 建議股各帳戶持有狀況，可直接加入下單 |
| 📋 比對矩陣 | 全部股票 × 全部帳戶持有狀況一覽表 |
| 🔴 缺少清單 | 各帳戶缺少的股票清單，可直接加入下單 |
| 👤 各帳戶 | 單一帳戶缺少清單 |

### 第四步：產生批次下單（選填）
點右上角「📋 批次下單範本」，或在缺少清單點「📋 全部加入下單」：

**永豐 CSV**
1. 輸入帳號（可新增多個）
2. 輸入或從清單帶入股票代號
3. 點框全選複製，貼入永豐批次下單介面

**國泰 XML**
1. 填入 DealAcc、DealAccUI、SSID
2. 輸入或從清單帶入股票代號
3. 下載 .xml 檔後匯入國泰下單系統
