# 📊 零股庫存監控工具

一個純前端的零股庫存比對工具，不需要伺服器、不需要安裝任何東西，直接用瀏覽器開啟即可使用。

🌐 **線上使用：** `https://roazhang.github.io/stock_monitor/`

---

## ✨ 功能介紹

| 頁籤 | 功能說明 |
|------|----------|
| 🔍 股票查詢 | 輸入股票代號或名稱，即時查詢各帳戶持有數量 |
| 📌 建議清單比對 | 上傳任意格式的建議清單，自動比對各帳戶缺哪些 |
| 📋 比對矩陣 | 所有股票 × 所有帳戶，一張表看清持有狀況 |
| 🔴 缺少清單 | 每個帳戶缺少的股票一覽 |
| 👤 各帳戶頁籤 | 單一帳戶的缺少清單 |

---

## 🚀 使用方式

### 第一步：上傳庫存檔案

- 支援格式：`.xlsx`、`.xls`、`.csv`
- 可一次上傳多個帳戶的庫存檔案
- 拖曳或點擊上傳皆可

**支援的庫存格式：**
- 欄位格式：`商品`（如 `1101 台泥`）+ `今餘`（股數）
- CSV 格式：`股票名稱`（如 `台泥(1101)`）+ 零股今餘欄

### 第二步：上傳建議清單（選填）

- 支援任意格式的 xlsx/csv
- 上傳後會自動偵測含有股票代號的欄位
- 點選正確的欄位後確認，即可進行比對
- 格式不同時只需重新選欄，**不需要修改程式**

### 第三步：開始比對

點擊「🔍 開始比對」，即可看到各帳戶的持有狀況與缺少清單。

## 🛠 技術說明

- 純前端，無需後端伺服器
- 使用 [SheetJS](https://sheetjs.com/) 解析 Excel / CSV 檔案
- 支援 Big5 / CP950 編碼（台股庫存檔案常見編碼）
- 所有資料僅在瀏覽器本地處理，**不會上傳至任何伺服器**

## 📝 注意事項

- 建議清單若有多個代號欄（如左右並排兩組），可同時選取多個欄位
- CSV 檔案請確保編碼為 Big5 或 UTF-8
- 股票名稱純英文（如 `LINEPAY`、`TPK-KY`）也能正確識別
